**Dungeon run console + player watch** (UNN-463/464/467; spatial M2 exploration).

The **DM console's** combat-vs-explore fork is decided once at its route loader (the DM's own actions re-run it); the **watch** has no server fork ‚Äî its phase is client-observable snapshot state.
